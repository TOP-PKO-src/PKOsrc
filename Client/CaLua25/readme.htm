==========================
= Building and installing
==========================
Prebuilt versions of Lua and CaLua in both debug and release form can be found in the 'lib' directory.


Halfway through development, I upgraded to Visual Studio .net from VC++ 6.0. If you can build Lua, you should be able to build CaLua from scratch. I've included pre-built versions of Lua and CaLua in both debug and release form, which should work for anyone running window.
